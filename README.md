# LosSimpsonsDex

Proyecto generado con [Angular CLI](https://github.com/angular/angular-cli) version 21.2.11.

## Servidor de desarrollo

Para iniciar un servidor local de desarrollo, ejecuta:

```bash
ng serve
```

Cuando el servidor este en marcha, abre el navegador y entra en `http://localhost:4200/`. La aplicacion se recargara automaticamente cada vez que modifiques un archivo fuente.

## Generacion de codigo

Angular CLI incluye herramientas para generar codigo de forma automatica. Para crear un componente nuevo, ejecuta:

```bash
ng generate component nombre-del-componente
```

Para ver la lista completa de esquemas disponibles, como `components`, `directives` o `pipes`, ejecuta:

```bash
ng generate --help
```

## Compilacion

Para compilar el proyecto, ejecuta:

```bash
ng build
```

Este comando compilara la aplicacion y guardara los archivos generados en el directorio `dist/`. Por defecto, la compilacion de produccion optimiza la aplicacion para mejorar rendimiento y velocidad.

## Pruebas unitarias

Para ejecutar las pruebas unitarias con [Vitest](https://vitest.dev/), usa:

```bash
ng test
```

## Pruebas de extremo a extremo

Para ejecutar pruebas de extremo a extremo (e2e), usa:

```bash
ng e2e
```

Angular CLI no incluye por defecto un framework de pruebas e2e. Puedes elegir el que mejor se adapte a tus necesidades.
